3-DOF Arduino Robotic Arm – Iteration 1

Mechanical Design | Embedded Systems | IR Remote Control

Objective Mechanical Design Electronics Software Assembly Media

Objective

This 3-DOF robotic arm project was designed to explore mechanical design, embedded systems, and servo-based control. The arm is controlled using an IR remote and includes a distance sensor to add environmental awareness for the gripper. This first iteration is simple, reliable, and ideal for learning the fundamentals of robotics.

Mechanical Design

  • 3D printed arm links (PLA)
  • Servo Horns:
    • Cross horn → Base rotation
    • Single-arm horns → Shoulder and Elbow
    • Modified double-arm horn → Gripper (both arms cut and shaped into rectangular tab for gear interface)
  • Gripper tension: Thick rubber band
  • Gripper attached to arm via M3 screw + nut

Key Design Note: The modified horn and hot glue bond create a simple, keyed interface to transfer torque efficiently while allowing easy rework.

Electronics & IR Remote Control

  • Arduino Nano controls all three servo joints and the gripper
  • IR Receiver module (pin 13) decodes commands from the remote
  • Ultrasonic distance sensor triggers automatic gripper action
  • Servo power: 5V 2A supply (separate from Arduino for reliability)

The IR remote allows precise joint movements: Up/Down → Shoulder, Left/Right → Elbow, Star/Hash → Base rotation, OK → Toggle gripper.

Software Architecture

  • Modular functions handle servo motion, IR input, gripper logic, and ultrasonic sensing
  • Servo angles stored as variables to allow smooth incremental movements
  • Auto-gripper mode triggers when ultrasonic sensor detects nearby objects
  • IR remote commands mapped to joint movements and gripper toggle

Example key function snippet:

void toggleGripper() {
    gripperOpen = !gripperOpen;
    if (!gripperOpen && !gripperNeedsOpenReset) {
        gripper.write(80);
        delay(500);
        gripper.write(0);
        delay(2000);
        gripperNeedsOpenReset = true;
    } else if (!gripperOpen && gripperNeedsOpenReset) {
        gripper.write(0);
    }
    if (gripperNeedsOpenReset && gripperOpen) {
        gripper.write(100);
        delay(250);
        gripper.write(90);
        gripperNeedsOpenReset = false;
        delay(2000);
    }
}

Step-by-Step Assembly

  1. 3D print all arm links and base components
  2. Attach servos to base, shoulder, and elbow positions
  3. Prepare servo horns (modify double-arm horn for gripper)
  4. Attach gripper to arm using M3 screw + nut, secure rubber band for tension
  5. Wire servos to Arduino Nano and connect IR receiver to pin 13
  6. Connect ultrasonic sensor (trig → pin 10, echo → pin 11)
  7. Upload Arduino code, calibrate joint angles, and test IR commands
  8. Test auto-gripper using ultrasonic sensor
